# Ronojit's Portfolio

A modern, responsive, and minimalist personal portfolio website built with a stylish "Bento Box" grid design. This project highlights a professional online presence, featuring sections for experience, tech stack, featured projects, location, work process, and client testimonials.

## ✨ Features

- **Modern Bento Box UI**: A structured, visually appealing, and organized grid layout.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports, transitioning gracefully from a complex grid to a single column on smaller screens.
- **Dynamic Animations**: Smooth scroll-reveal animations using the Intersection Observer API for an engaging user experience.
- **Interactive Elements**: Micro-animations on hover states for buttons, links, and cards.
- **Clean Aesthetic**: A soft, light color palette with high-contrast text and subtle accents inspired by modern Apple-like design principles.
- **Custom Graphics**: Inline avatar typography and stylized mockups.
- **No Build Step Required**: Built with Vanilla HTML, CSS, and JS for maximum simplicity and zero dependencies.

## 🛠️ Technology Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom properties (variables), CSS Grid, Flexbox, and advanced animations.
- **JavaScript (ES6+)**: Vanilla JS for intersection observers and interactive elements.
- **Phosphor Icons**: Lightweight, clean iconography used throughout the design.
- **Google Fonts**: Utilizes the 'Inter' font family for crisp, modern typography.

## 💻 Getting Started

### Prerequisites

You only need a modern web browser to view this project locally.

### Installation & Running Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/ronojitmondalin-freelancer/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your favorite browser:
   - On Windows: Double-click `index.html` or use `start index.html` in the command line.
   - On macOS: `open index.html`
   - On Linux: `xdg-open index.html`
   - Alternatively, use a local server like Live Server (VS Code extension) or Python's HTTP server (`python -m http.server`) for a better development experience.

## 🎨 Customization (Theme Variables)

The design is heavily driven by CSS Variables defined in `:root` inside `styles.css`. You can easily tweak the look and feel by modifying these values:

```css
:root {
    --bg-color: #f7f7f9;
    --card-bg: #ffffff;
    --text-main: #1d1d1f;
    --text-muted: #86868b;
    --accent-orange: #ff6b00;
    --accent-green: #34c759;
    /* ... and more spacing/border variables */
}
```
